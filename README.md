# x2x5 博导技能

本地网页工具，用于管理和复用科研流程 skills 卡片。

## 卡片分区

- **主力卡片**：常用卡片，支持拖拽排序，可折叠
- **阶段卡片**：按科研流程 7 个阶段分组，每阶段独立折叠面板
  1. 选题与调研
  2. Idea 构思
  3. 方法设计
  4. 实验执行
  5. 论文写作
  6. 审稿与修改
  7. 投稿与准备
- **垃圾卡片**：存放用户自建后删除的卡片，支持清空

## 功能

- 一键复制模板 + 输入内容
- 使用次数统计（复制且输入非空时 +1）
- 按次数排序 / 次数清零
- 新增卡片：粘贴 AI 返回内容自动建卡
- 卡片编辑与删除

## 数据源

卡片模板读取 `skills.md`，格式：

```markdown
# Part I: 科研流程 Skills 卡片

### 阶段 1：选题与调研
## 卡片标题
​```markdown
...prompt 内容...
​```
```

- `### ` 开头 → 阶段分类
- `## ` 开头 → 卡片标题
- 代码块 → prompt 正文

用户操作数据（主力排序、新增卡片、使用次数等）存储在浏览器 `localStorage`。

## 本地启动

```bash
python3 -m http.server 8080
```

打开 http://localhost:8080

## 文件说明

- `index.html` — 页面结构
- `styles.css` — 样式
- `app.js` — 交互逻辑
- `skills.md` — 卡片模板数据
