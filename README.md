# Card

本地网页工具，用于管理和复用科研流程 skills 卡片。支持中英文双数据源同时加载，切换语言时用户状态（输入内容、使用次数、编辑记录）不丢失。

## 卡片结构

卡片按科研流程 7 个阶段分组，每阶段独立折叠面板：

1. 选题与调研
2. Idea 构思
3. 方法设计
4. 实验执行
5. 论文写作
6. 审稿与修改
7. 投稿与准备

每个卡片包含：
- **标题**：卡片名称
- **Hint**：输入框提示，放在 Markdown 中每个卡片的 prompt 下方
- **Prompt**：技能模板内容

## 功能

- 一键复制模板 + 输入内容
- 使用次数统计（复制且输入非空时 +1）
- 次数清零
- 新增卡片：粘贴 AI 返回内容自动建卡
- 卡片编辑与删除
- 界面语言切换（中文/英文）：只改按钮和标签文字
- 技能语言切换（中文/英文）：切换 prompt 模板和 hint 内容，用户状态保留

## 数据源

同时加载 `skills.md`（中文）和 `skills-en.md`（英文），用统一 ID（阶段-序号，如 `1-1`、`2-3`）索引。每个 ID 下存储中英文两个版本的卡片数据。

卡片模板格式：

```markdown
## 阶段 1：选题与调研

### 文献检索策略

````markdown
# Role
...prompt 内容...

# Input
[在此处粘贴你的研究方向、关键词或简要描述]
````

hint: 粘贴你的研究方向、关键词或简要描述

---
```

- `## ` 开头 → 阶段分类
- `### ` 开头 → 卡片标题
- ````markdown` 代码块 → prompt 正文
- `hint:` 行 → 输入框提示（放在代码块下方）

用户操作数据（编辑记录、新增卡片、使用次数等）存储在浏览器 `localStorage`。

## 本地启动

```bash
python3 -m http.server 3000
```

打开 http://localhost:3000
