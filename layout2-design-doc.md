# 布局 2 设计文档：科研卡片流程画布（定稿）

## 一、概述

布局 2 是一个 draw.io 风格的流程画布。用户从右侧卡片抽屉中选取卡片，拖到左侧画布上，用箭头连接，搭建属于自己的科研流程。

---

## 二、已确认的设计决策

| 决策点 | 结论 |
|-------|------|
| 触发方式 | 右上角按钮区新增「切换布局」按钮，点击后全屏切换到布局 2 |
| 布局关系 | 完全切换，布局 1 和布局 2 互不干扰 |
| 画布位置 | 左侧（大面积） |
| 卡片池位置 | 右侧（窄栏，抽屉式折叠） |
| 画布节点是否需要输入框 | **不需要**，画布是纯视觉编排工具 |
| 同一卡片能否多次出现 | **允许**，比如 3 篇论文各一个「精读笔记」节点 |
| 主标题能否修改 | **不能** |
| 副标题能否修改 | **能**，本地修改（副标题当前是阶段名，确实冗余） |
| 箭头是否携带信息 | **不携带**，纯视觉顺序关系 |
| 目标风格 | draw.io —— 左边抽屉选图形，右边画布搭流程 |

### 右侧卡片池：抽屉式

每个阶段是一个可折叠的抽屉（`<details>` + `<summary>`）：

```
▶ 阶段 1：选题与调研          ← 点击展开
    [文献检索策略]             ← 可拖拽的卡片项
    [论文精读笔记]
    [领域图谱梳理]

▶ 阶段 2：Idea 构思
    [研究 Idea 头脑风暴]
    [可行性评估]
    [创新点提炼]

▶ 阶段 3：方法设计
    ...

▶ 阶段 4：实验执行
    ...

▶ 阶段 5：论文写作
    ...

▶ 阶段 6：审稿与修改
    ...

▶ 阶段 7：投稿与准备
    ...
```

默认全部折叠。用户点击展开某个阶段，然后把卡片拖到画布上。

### 画布上的节点

每个节点包含：
- **主标题**（不可修改）：卡片名称，如「论文精读笔记」
- **副标题**（可修改）：默认显示阶段名，双击可编辑，本地生效
- **底部连接点**：拖出箭头
- **顶部连接点**：拖入箭头

```
    ●  ← 顶部连接点（接收箭头）
┌───────────────┐
│  论文精读笔记   │  ← 主标题（不可改）
│  精读那篇CVPR  │  ← 副标题（可改，默认=阶段名）
└───────────────┘
    ●  ← 底部连接点（发出箭头）
```

## 四、交互设计

### 4.1 添加节点

1. 右侧抽屉中，鼠标按下某张卡片不松手
2. 拖动到左侧画布区域
3. 松手，节点出现在该位置
4. 如果直接点击（非拖拽），节点出现在画布中央

### 4.2 移动节点

- 鼠标按住节点主体区域拖动
- 节点跟随鼠标移动
- 松手后节点停在新位置
- 所有连接到该节点的箭头实时更新

### 4.3 编辑副标题

- 双击节点的副标题文字
- 变成可编辑的 input
- 按 Enter 或点击别处确认
- 按 Escape 取消

### 4.4 画箭头

1. 鼠标按下节点 A 的底部连接点
2. 拖动，出现一条临时跟随鼠标的线
3. 拖到节点 B 的顶部连接点上松手
4. 形成一条从 A 到 B 的箭头
5. 如果拖到空白处松手，取消（线消失）

箭头用平滑的贝塞尔曲线绘制，从 A 底部出发，经过适当弯曲，到达 B 顶部。

### 4.5 删除箭头

- 点击箭头线条，选中（高亮显示）
- 按 Delete 键删除
- 或者右键菜单 → 删除

### 4.6 删除节点

- 点击节点，选中（蓝色边框高亮）
- 按 Delete 键删除
- 或者右键菜单 → 删除
- 删除节点时，所有连接到该节点的箭头一并删除

### 4.7 画布滚动

- 画布区域比视口大（默认 4000×3000px）
- 鼠标滚轮上下滚动，Shift+滚轮左右滚动
- 或者按住画布空白区域拖动平移

## 六、箭头绘制方案

使用 SVG 覆盖层。画布的 HTML 结构：

```html
<div class="canvas-scroll-container">
  <div class="canvas-inner" style="width: 4000px; height: 3000px; position: relative;">

    <!-- SVG 箭头层 -->
    <svg class="canvas-arrows" style="position: absolute; inset: 0; width: 100%; height: 100%; pointer-events: none;">
      <defs>
        <marker id="arrowhead" markerWidth="10" markerHeight="7"
                refX="10" refY="3.5" orient="auto">
          <polygon points="0 0, 10 3.5, 0 7" fill="#6c757d" />
        </marker>
      </defs>
      <!-- 每条箭头是一个 path -->
      <path class="canvas-arrow" d="M..." marker-end="url(#arrowhead)"
            style="pointer-events: stroke;" />
    </svg>

    <!-- 节点层 -->
    <div class="canvas-node" style="position: absolute; left: 200px; top: 150px;">
      <div class="node-connector-top"></div>
      <div class="node-title">论文精读笔记</div>
      <div class="node-subtitle">精读那篇CVPR</div>
      <div class="node-connector-bottom"></div>
    </div>

  </div>
</div>
```

箭头的 path 计算：
- 起点：节点 A 底部连接点的中心坐标
- 终点：节点 B 顶部连接点的中心坐标
- 控制点：起点和终点的垂直中点，水平偏移使曲线自然

```javascript
function computeArrowPath(fromNode, toNode) {
  const fromX = fromNode.x + fromNode.width / 2;
  const fromY = fromNode.y + fromNode.height;
  const toX = toNode.x + toNode.width / 2;
  const toY = toNode.y;
  const midY = (fromY + toY) / 2;
  return `M ${fromX} ${fromY} C ${fromX} ${midY}, ${toX} ${midY}, ${toX} ${toY}`;
}
```

## 八、实现分步

### Step 1：布局切换 + 骨架
- 右上角加「切换布局」按钮
- 点击后隐藏布局 1，显示布局 2
- 布局 2 基本骨架：左侧画布容器 + 右侧卡片抽屉栏
- 再点切回布局 1

### Step 2：卡片抽屉
- 右侧按 7 个阶段构建折叠抽屉
- 每个抽屉里列出该阶段的卡片名称
- 点击卡片名称 → 在画布中央生成一个节点

### Step 3：画布节点
- 节点渲染：主标题 + 副标题 + 顶部/底部连接点
- 节点可拖拽移动
- 双击副标题可编辑
- Delete 键删除节点

### Step 4：箭头连线
- 从底部连接点拖出临时线
- 拖到顶部连接点松手形成箭头
- 箭头用贝塞尔曲线绘制
- 节点移动时箭头实时更新
- 点击箭头选中 + Delete 删除

### Step 5：状态持久化
- 画布状态（节点+箭头）存入 localStorage
- 页面加载时恢复

### Step 6：打磨
- 画布平移（拖拽空白区域或滚轮）
- 抽屉拖拽到画布（直接拖，不只点击）
- 视觉细节调整
